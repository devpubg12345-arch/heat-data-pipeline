# Heat Equation Data Pipeline

## Overview
This project demonstrates how to generate and process data from a heat equation using numerical methods.

## What I did
- Solved heat equation using finite difference method
- Generated dataset (x, t, u)
- Processed data using pandas
- Normalized and analyzed features

## Files
- data_processing.ipynb → main work
- heat_data.csv → dataset

## Tools Used
- Python
- NumPy
- pandas
- matplotlib
- scikit-learn
